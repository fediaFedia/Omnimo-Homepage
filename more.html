How to Install

If you need help installing or uninstalling Omnimo, follow this simple guide



Overview: Installing Omnimo 6.1
1. Download Rainmeter
2. Install and open Rainmeter
3. Download\Extract Omnimo
4. Double Click and follow setup



Uninstalling Omnimo 6.1
Click the Start button on your computer, All Programs > Omnimo UI. Or search for "Uninstall" in your startmenu.



Alternatively, you can navigate to Documents \ Rainmeter \ Skins and delete the WP7 folder (that's where Omnimo is stored).
